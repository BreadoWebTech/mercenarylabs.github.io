Variables

What are Variables?

Imagine variables as little boxes you can store information in, and that you can open these boxes whenever you want to get the information back you put in it before. That’s literally how variables work. You create variables in two simple steps: Declaration: Creating a variable, you do that by giving it a name and defining its type (In python you don’t need to define its type, since the type of the variable changes, depending on what value you assign to it, automatically). Initialization: assign a value to that variable. You can create a variable in Python like this: 
x = 10
This already includes both steps mentioned. 
Declaration ( giving it its name ( x ) )
Initialization ( assign the value to it ( 10 ) and assign the type of it ( in this case int, since python automatically knows if you put an int, string or whatever type there )
If you read carefully, you might notice that assigning the type slipped into the Initialization part. That’s because Python dynamically updates the type of the variable on its own, depending on what the value is. If you create a variable named x and assign a value with the type string to it, python would automatically set the type of the variable to string. If you chose to assign a new value to x , maybe an int, it would actually discard the old variable, and create a new one with the name and the new type int. I will show you an example of that here: 
x = 10
print(x)
x = "hi"
print(x)
If you run this program you will notice that you were able to reassign a value to the same variable with a different type. Theres a function in python you can use to get the type of a variable, called the type() function. 
x = 10
print(type(x))
x = "hi"
print(type(x))
You can see, that the type of the variable automatically changes just by assigning different types of values to that variable.

Functions

What are functions?

essentially, you can see them as a block of code, that you mark as a huge block with a name. If you call that name of this block in the program, the stuff in this block gets executed without the need of retyping the same code again. That’s just a function. A block of code you can reuse. A function can also take arguments. Depending on what code is in the function, youll need the have the function take arguments. Heres a quick example of how to create a function: 
def myFunc():
    # btw, stuff with hashtags infront of em are comments in python
    # in here goes the code of the function
    print("hi")
That’s a very simple example of a function To use it, youll have to call it. Calling a function is just putting its name ( here myFunc ) followed by these: () like this: 
myFunc()
So the complete python program to write and call a function would look like this: 
def myFunc():
    # btw, stuff with hashtags infront of em are comments in python
    # in here goes the code of the function
    print("hi")
myFunc()
                                              
#Output will be "hi"

def calc_rect(length, width):

    return area

It takes two arguments, length, width, which we fill with the vars rect_length and rect_with.
It then takes these values, does the calculation, an returns the area, which we store in the result var and print it out.
inside those () you can decide if a function takes arguments. If it doesn’t, just leave it empty.
The order in which you put the arguments there matters.
if you have calc_rect take width and then length, and don’t change the order in calling the function, its gonna assign rect_length to width and rect_width to length

For Loops

You can use a for loop to iterate over something ( e.g. a list ). it loops trough that something for every item that exists in it, thus named "for loop" if you have a list, called myList, with 2 items in it, and want to print each item out, without having to use this annoying index number stuff, youll use a for loop: 
myList  = [1,2]
for i in myList:
    print(i)
this is gonna iterate over every item in the list myList, and print each item out.
i is a variable name and can be replaced with any var name, but i is just short for item spoken, a for loop is this:
for every item in myList: print item
that's it.
you can also make the for loop do something for a specific amount of times. 
for var in range(1,11):
    # do something 10 times
That’s it. These a for loops. Pretty easy, right?

While loops

While loops are pretty easy too.

its just:
While this is True:
do something
and if for some reason it isn’t true anymore
break out of the while loop
A while loop is defined like this: 
while condition:
    # do something
as long as the condition is true, it will do everything in that code block, and repeat it. As an example, if you had a while loop, that is defined like this: 

x = 1
while x == 1:
    # btw "== is just saying "the same as
    print("1")
this will run forever, because x never changes to another value,so there will be printed out 1 till infnity
if the value of x would change inside the while loop, it would break out of the loop. 

x = 1
while x == 1:
    print("1")
    x = 5
you can also use this: 

while True:
    # do something
usally used if the developer wants something to always run, no matter what.
These are while loops. Also pretty easy right?
